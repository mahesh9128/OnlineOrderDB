Query : Retrieve the list of restaurants of a particular area and of particular cuisine!

SQL : SELECT R.restId,R.name,R.cuisine,R.status,R.address,RT.openTime,RT.closeTime FROM `dbproject`.`restaurant` as R, `dbproject`.`rest_timings` as RT WHERE R.restId=RT.restId and address like 'area' and R.cuisine='cuisine' order by R.name ;

Query : Display all the Restaurants with their cuisine and open at the “current time”!

SQL : SET time_zone = '-06:00'; SELECT R.restId,R.name,R.cuisine,R.status,R.address,RT.openTime,RT.closeTime FROM `dbproject`.`restaurant` as R, `dbproject`.`rest_timings` as RT WHERE R.restId=RT.restId and R.cuisine='cusine' and CURTIME() BETWEEN RT.openTime and RT.closeTime;

Query : View all the menu items of a restaurant with particular cuisine having the combo prices less than given price.

SQL : SELECT R.name,M.comboId,M.appetizer,M.mainCourse,M.dessert,M.price FROM `dbproject`.`restaurant` as R,`dbproject`.`menu` as M WHERE R.restId=M.restId and R.cuisine='cuisine' GROUP BY M.comboId HAVING M.price< price;

Query : View the discount on a Combo ID of any restaurant on a particular day of the week.

SQL : SELECT R.name,D.comboId,D.discount FROM `dbproject`.`restaurant` as R,`dbproject`.`discount` as D WHERE R.restId=D.restId and R.name="restaurant name" and D.comboId= "comboID" and D.day='day of the week';

Query : View the overall (average) rating of restaurants.

SQL : SELECT R.name as RESTAURANT,avg(rating) as OVERALL_RATING FROM `dbproject`.`restaurant` as R,`dbproject`.`review` as RV WHERE R.restId=RV.restId GROUP BY R.name ORDER BY OVERALL_RATING;

Query : Display the restaurant with highest ratings.

select RESTAURANT_NAME,Rating from `dbproject`.`high_rating_view` where Rating in (SELECT max(Rating) from `dbproject`.`high_rating_view`);

Query : View all the comments and the ratings given by customers about a particular restaurant.

SQL : SELECT RV.custId,R.name,RV.comment,RV.rating FROM `dbproject`.`restaurant` as R,`dbproject`.`review` as RV WHERE R.restId=RV.restId and R.name = 'restaurant name' GROUP BY RV.custId;
