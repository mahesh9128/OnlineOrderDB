Query : View the customer's address with a particular order number.

SQL : SELECT C.fname,C.address FROM `dbproject`.`customer` as C,`dbproject`.`order` as O WHERE C.custId=O.custId and O.orderNum="order num";

Query : Find all orders whose status is “pending” and get its count.

SQL : SELECT COUNT(*) as No_of_pending_orders FROM `dbproject`.`order` as O WHERE O.status='pending';

Query : View all the orders that are pending for delivery to customer who lives in particular area.

SQL : SELECT O.comboId,O.custId,O.orderNum,O.restId FROM `dbproject`.`customer` as C,`dbproject`.`order` as O WHERE C.custId=O.custId and C.address like 'area' and O.status='pending';
